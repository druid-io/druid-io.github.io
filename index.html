Apache Druid (incubating) is a high performance analytics data store for event-driven data.

Download GitHub

Overview

Analyze event streams

Druid provides fast analytical queries, at high concurrency, on both historical and real-time data. Druid is often used to power interactive UIs.

Build next generation data architecture

Druid works out-of-the-box with data lakes (HDFS, AWS S3, etc) and message buses (Kafka, AWS Kinesis, etc). Druid is used as a query layer in next-gen streaming and batch data stacks.

Unlock new workflows

Druid is designed for rapid, ad-hoc analytics on both real-time and historical data. Explain trends, explore data, and quickly iterate on queries to answer questions.

Deploy anywhere

Druid can be deployed in any *nix environment on commodity hardware, both in the cloud and on premise. Druid is cloud-native: scaling up and down is as simple as adding and removing processes.
